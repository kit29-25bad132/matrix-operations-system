# 📌 Matrix Operations System

## 📖 Overview

The Matrix Operations System is a Python-based command-line application designed to perform fundamental matrix computations. It allows users to input matrices and execute operations such as addition, subtraction, and multiplication through a simple interactive interface.

This project demonstrates core programming concepts including loops, functions, input validation, and modular code design.

---

## 🎯 Objectives

* Implement basic matrix operations from scratch
* Understand matrix computation logic
* Practice modular programming in Python
* Build a user-driven CLI (Command Line Interface)

---

## ⚙️ Features

* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Input validation for correct matrix dimensions
* Menu-driven interface for user interaction

---

## 🛠️ Tech Stack

* Language: Python
* Concepts Used:

  * Functions
  * Nested loops
  * Conditional statements
  * List data structures

2. Navigate to the project folder:
